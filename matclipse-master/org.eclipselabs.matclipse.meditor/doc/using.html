Using Meditor

Meditor extends a standard Eclipse Source Editor, which provides many functionalities. This help section describes the features brought in by Meditor.

Matlab help access

If the cursor is positioned at a Matlab function, the (HTML) Matlab help page for this function can be shown by:

  • pressing F1
  • clicking Help > Matlab help

Indentation

When beginning a new line, Meditor automatically sets the correct indentation for the new line and, if necesary, corrects the precedent line's indentation.

It's also possible to force the correct indentation of one or more lines manually:

  • The simplest way to do this is to press tab somewhere before the first non whitespace character in the line to modify
  • Pressing Ctrl-I indents all selected lines (or the line the cursor is positioned in) correctly
  • The same effect has the command Edit > Correct indentation
  • or in the context menu Source > Correct indentation

To indent the whole document one can:

  • Press Ctrl-Shift-I
  • Click Edit > Indent document
  • or in the context menu Source > Indent document

(Un-)Commenting

There are two commands to comment or uncomment all selected lines. They can be accessed through the menu or the context menu.

  • Menu: Edit > Comment and Edit > Uncomment
  • Context menu: Source > Comment and Source > Uncomment

Export functions

One can export the editor's content to a XML, HTML, LaTeX and/or PDF file. There are two different ways to do this:

File > Export as...  Opens a "Save as" Dialog where you specify the filename. The file extension defines the output format to use. All other options are taken from the export preferences.
File > Export This opens a Dialog with some options, e.g. the output formats, the file's title, ...
Some more options are taken from the export preferences. This command creates all specified output files. The filename (without extension) is taken from the matlab file. Each export creates a hidden file in the directory of the source file where the data from the export dialog is stored in. If one exports the same file again, this data will be loaded.
