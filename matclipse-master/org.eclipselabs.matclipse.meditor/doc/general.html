General

Meditor is a Matlab editor that provides support for writing matlab code, getting help and exporting code to several output formats.

Features

These are Meditor's main features:
  • Syntax highlighting
  • Automatic indentation
  • Visualization of matching brackets
  • Matlab help access
  • Export of Matlab code, extended by the possibility to use formatted text
